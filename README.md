# File-Compress_Rust
🦀 File Compress Utility - Rust

A lightweight and efficient file compression utility built with Rust that allows users to compress files while leveraging Rust’s speed, reliability, and memory safety.

Features

✅ Compress files efficiently
✅ Fast command-line execution
✅ Error handling for invalid or missing files
✅ Lightweight and performance-focused implementation
✅ Safe systems programming with Rust

⸻

Project Overview

This project was built to explore practical systems programming concepts in Rust and gain hands-on experience with file operations and compression techniques.

The application processes files through a simple command-line interface and creates compressed output while maintaining performance and safety.

Installation

Clone the repository:

git clone https://github.com/Hellboy28D/File_Compress_Rust.git

Move into the project directory:

cd File_Compress_Rust

Build the project:

cargo build

⸻

Usage

Run the project:

cargo run -- <filename>

Example:

cargo run -- sample.txt

Output:

Compressing file...
Processing complete
Compressed successfully
